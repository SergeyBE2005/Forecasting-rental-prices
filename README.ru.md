# Прогнозирование арендных цен

**Forecasting-rental-prices**

Этот проект демонстрирует конвейер машинного обучения для прогнозирования стоимости жилья. В нем используется набор данных о жилье в Калифорнии для обучения модели Random Forest Regressor и оценки ее производительности. Обученная модель сохраняется для будущих прогнозов.

## Цель проекта

Основная задача — построить и обучить регрессионную модель для прогнозирования медианной стоимости дома в определенном квартале Калифорнии на основе различных признаков, таких как медианный доход, возраст дома и среднее количество комнат.

## Структура проекта

Репозиторий содержит следующие ключевые файлы:

- **Forecasting-rental-prices.ipynb**: Основной Jupyter Notebook, содержащий весь рабочий процесс, включая:
  - Загрузку данных с использованием sklearn.datasets.fetch_california_housing
  - Исследовательский анализ данных (EDA) с тепловыми картами корреляции и точечными графиками
  - Предобработку данных и отбор признаков
  - Обучение модели с использованием Random Forest Regressor
  - Оценку модели с помощью коэффициента R2
  - Код для сохранения обученной модели
- **model.pkl**: Сохраненная предварительно обученная модель Random Forest
- **requirements.txt**: Список всех зависимостей Python, необходимых для запуска ноутбука
- **dockerfile**: Dockerfile для создания контейнеризированного окружения проекта
- **correlation.png**: Сохраненная визуализация матрицы корреляции
- **scatter.png**: Сохраненный точечный график, показывающий связь между медианным доходом и ценой
- **README.md**: Этот файл (на английском языке)
- **README.ru.md**: Русская версия этого файла

## Начало работы

### Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- (Опционально) Docker для запуска проекта в контейнере

### Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/SergeyBE2005/Forecasting-rental-prices.git
   cd Forecasting-rental-prices
   ```

2. Создайте виртуальное окружение (рекомендуется):
   ```
   python -m venv venv
   source venv/bin/activate  # В Windows используйте venv\Scripts\activate
   ```

3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

### Запуск ноутбука

Для изучения кода и обучения модели:

```
jupyter notebook Forecasting-rental-prices.ipynb
```

Выполните ячейки ноутбука последовательно для:
1. Загрузки и исследования данных о жилье в Калифорнии
2. Визуализации корреляций признаков и взаимосвязей
3. Обучения модели Random Forest
4. Оценки точности модели
5. Сохранения обученной модели в файл model.pkl

### Запуск с Docker

Для сборки и запуска проекта в контейнеризированном окружении:

1. Соберите Docker-образ:
   ```
   docker build -t forecasting-rental-prices .
   ```

2. Запустите контейнер:
   ```
   docker run -p 8888:8888 forecasting-rental-prices
   ```
   Это запустит сервер Jupyter Notebook внутри контейнера. Откройте его в браузере по адресу http://localhost:8888

## Производительность модели

Текущая модель Random Forest Regressor со 100 деревьями достигает коэффициента R2 примерно 0.8141 на тестовом наборе данных. Это означает, что модель объясняет около 81.4% дисперсии целевой переменной (цен на жилье).

## Использование

Для использования сохраненной модели для прогнозирования на новых данных:

1. Убедитесь, что у вас есть файл model.pkl
2. Используйте следующий Python-скрипт в качестве примера:

```python
import joblib
import numpy as np

# Загрузка модели
model = joblib.load('model.pkl')

# Пример: прогноз цены для нового образца (8 признаков из набора данных)
# [MedInc, HouseAge, AveRooms, AveBedrms, Population, AveOccup, Latitude, Longitude]
new_data = np.array([[8.0, 45.0, 6.0, 1.0, 1000.0, 3.0, 37.0, -122.0]])
predicted_price = model.predict(new_data)

print(f'Прогнозируемая цена: {predicted_price[0]:.2f}')
```
